About This Project

This application was built to demonstrate end-to-end deployment of a machine-learning model inside a lightweight Flask web application.

Dataset

The model was trained on the publicly available Kaggle Credit Card Fraud Detection dataset, which contains 284,807 transactions, 492 of which are fraudulent.

Model

A logistic-regression / random-forest classifier (placeholder – replace with actual model name) was selected after cross-validation. Features were scaled with StandardScaler and the class imbalance was handled using SMOTE oversampling.

Tech Stack

  • Backend: Python 3.11, Flask 3
  • ML: scikit-learn, pandas, numpy
  • Frontend: HTML5, CSS3, vanilla JavaScript
